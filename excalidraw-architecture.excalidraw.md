## Color Scheme Reference

```css
/* GastroMap Apple Design System */
:root {
  /* Primary Colors */
  --apple-bg: #f5f5f7;           /* Light gray background */
  --apple-dark: #1a1a1d;         /* Dark text / primary content */
  --apple-accent: #ff9500;       /* Apple-style orange (primary CTA) */
  
  /* Helper Colors */
  --apple-green: #34c759;        /* Success, active states */
  --apple-red: #ff3b30;          /* Error, delete actions */
  --apple-gray: #8e8e93;         /* Secondary text */
  --apple-gray-light: #afafaf;   /* Tertiary text */
  
  /* UI Elements */
  --card-shadow: 0 4px 12px rgba(0,0,0,0.08);
  --header-height: 74px;
  --bottom-nav-height: 68px;
  --detail-sheet-height: 65%;    /* Bottom sheet expanded */
  --detail-page-overlay: rgba(0,0,0,0.5);
}
```

## Component Interaction Flow

```
[User Opens App]
        ↓
[Bottom Nav: Explore Tab Active]
        ↓
[User Browses Restaurants]
        ↓
    ┌─────────────────────┐
    │ Tap Restaurant Card │
    └─────────────────────┘
             ↓
    ┌─────────────────────┐
    │ Detail Sheet Opens  │
    │ (Bottom Slide Up)   │
    └─────────────────────┘
             ↓
    ┌─────────────────────┐      ┌─────────────────────┐
    │ See Restaurant Info │      │ View Map Preview    │
    └─────────────────────┘      └─────────────────────┘
             ↓                              ↓
    ┌─────────────────────┐      ┌─────────────────────┐
    │ Tap "See Details"   │      │ Tap Map Marker      │
    └─────────────────────┘      └─────────────────────┘
             ↓                              ↓
    ┌────────────────────────────────────────────────────┐
    │          Full Detail Page Opens (Full Screen)      │
    ├────────────────────────────────────────────────────┤
    │  Slides: Carousel Navigation                       │
    │  Tabs: Description | Menu | Reservation | Reviews │
    │  Reservation: Table Select + Date/Time Picker      │
    └────────────────────────────────────────────────────┘
             ↓
    ┌─────────────────────┐
    │ Tap "Забронировать" │
    └─────────────────────┘
             ↓
    [Booking Confirmed - Sheet/Detail Page Closes]
```

## Responsive Breakpoints

- Mobile: 320px - 480px (primary target)
- Tablet: 768px - 1024px (secondary target)
- Desktop: 1024px+ (desktop view with larger container)

## File Structure Summary

```
gastromap/
├── index.html                    # Main HTML structure with all sections
├── styles/
│   └── main.css                 # 500+ lines of Apple-style CSS
├── scripts/
│   └── app.js                   # Vanilla JS (233 lines)
│       ├── initNavigation()     # Bottom tab switching
│       ├── initRestaurantCards() # Card click handlers
│       ├── initMapMarkers()     # Map marker click handlers
│       ├── initMapToggle()      # Cards/Map view toggle
│       ├── openDetailSheet()    # Bottom sheet logic
│       ├── initDetailPage()     # Full page logic
│       ├── initCarousel()       # Image slider logic
│       ├── initTabs()           # Detail tab switching
│       └── initReservation()    # Table + date/time logic
└── assets/
    ├── images/                  # Restaurant photos
    └── icons/                   # SVG icons (optional)
```
